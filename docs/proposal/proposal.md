# Sonic Hub World - Project Proposal

This document is the required project proposal for ICG, made by Duarte Branco (nº119253).

## Project Overview

Ths project proposes the development of a open 3D hub world inspired by the known character \textit{Sonic the Hedgehog}. This concept was already implemented in the original \textit{Sonic Jam} (1997) game for the Sega Saturn, so this project will be heavily inspired by it.

In the original game, the hub world served merelly as a place where the player could roam around and enter the rooms where he wouldplay the actual games. Yet, in this project, the hub will only allow the player to roam around an explore the scenery, without any particular goal. I might add "checkpoints" throuout the map, so the player could play mini-races (go from checkpoint A to checkpoint B), but it would not be the sole purpose of the game.

## Scene & 3D Modeling

The world will be a typical Sonic enviroment, resembling the Green Hill Zone (most iconic Sonic zone). It will have varied terrain, including grassy hills, paths and water zones. Key necessary models include: trees, rings, flowers, walls, water, enemy and Sonic. For these models, everyone but Sonic will be made entirely by me using only three.js (not blender). The Sonic model will be one made by someone else (will give credit), but, the animations (running, walking, jumping, rolling) and the face emotions (smiling, amazed, serious) will still be done by me.

The user will be able to control Sonic with these controls:

 - *Space* - jump
 - *Shift* - spin dash
